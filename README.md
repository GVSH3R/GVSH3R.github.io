# Portafolio Profesional - Gustavo Hernández

Repositorio oficial del sitio web personal y portafolio de Gustavo Hernández, Software Engineer especializado en desarrollo Full Stack y Mobile.

## Perfil Profesional
Ingeniero con amplia experiencia en la creación de soluciones de software escalables. Especializado en la implementación de arquitecturas sólidas como Clean Architecture y MVVM, con un enfoque particular en sectores de alta exigencia como Fintech y Retail.

## Especificaciones Técnicas
El portafolio y los proyectos presentados integran las siguientes tecnologías:

* Backend: Java, Spring Boot, Python, JPA, Maven.
* Mobile: Dart (Framework Flutter), Kotlin (Android SDK), Jetpack Compose.
* Frontend: JavaScript (ES6+), HTML5, CSS3, Bootstrap 5.
* Bases de Datos: MySQL, Oracle.
* Herramientas de Desarrollo: Git, GitHub, Postman, VS Code.

## Proyectos Destacados

### TaxiDiver
Aplicación de movilidad diseñada para la gestión logística y solicitud de viajes en el AIFA. Desarrollada con tecnología multiplataforma para optimizar la geolocalización en tiempo real.

### Grid & Flex
Plataforma de comercio electrónico desarrollada mediante metodología ágil Scrum en un equipo multidisciplinario, enfocada en la persistencia de datos y experiencia de usuario.
